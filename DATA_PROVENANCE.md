# Data Provenance

The seed file `data/publications_context.csv` was parsed from copied profile/publication text supplied in the working context. It covers publication-like records visible in that copied text. The original profile may contain more publications than the copied context includes.

The app supports additional manual imports from copied publication-profile text, institutional pages, manually prepared CSV, ORCID/BibTeX-converted records, Google Scholar / Publish or Perish CSV exports, and ResearchGate-style copied publication lists.

The project does not automatically scrape restricted academic profile pages.
